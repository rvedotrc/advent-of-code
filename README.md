# Advent of Code 2021

https://adventofcode.com/2021

In Ruby.

```
bundle
bundle exec rspec
````

To run with your test data,
```
mkdir -p input
````
then save each day's input to `input/day1`, `input/day2` and so on.

Then use `bundle exec ./bin/test DAY PART`, e.g. for day 3 part 1,
```
bundle exec ./bin/test 3 1
```
